Information about "3-point Figures Building" app

About

3-point Figures Building application is an application that builds parallelogram using three points pointed by user with left mouse click on bordered area. Also it builds a circle that has the same area and mass center as the parallelogram.

User can see the coordinates of points that he pointed and the area of the builded figures on the right side bar and change it by moving any of those points.

User also can start all the process from the very beginning clicking Reset button.

How to use

  • point three times on bordered area by left mouse clicking - you will have three point as result;
  • application will build parallelogram and circle using those points automatically;
  • move any of your points to change it coordinates and the figures will rebuild;
  • press Reset button to clear the bordered area and start all the process from the very beginning.
